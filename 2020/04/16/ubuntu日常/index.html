代理工具 （proxychain）

## 安装 proxychains
sudo apt-get install proxychains
报错：ERROR: ld.so: object ‘libproxychains.so.3’ from LD_PRELOAD cannot be preloaded (cannot open shared object file): ignored.

解决方法：

## 搜索 libproxychains.so.3 文件
➜  ~  find /usr/ -name libproxychains.so.3 -print
/usr/lib/x86_64-linux-gnu/libproxychains.so.3

## 编辑 /usr/bin/proxychains 文件
export LD_PRELOAD=libproxychains.so.3
改为
export LD_PRELOAD=/usr/lib/x86_64-linux-gnu/libproxychains.so.3

运行报错

解决方法：

sudo apt install libgconf-2-4
报错：Failed to load module “canberra-gtk-module”

解决方法：

sudo apt install libcanberra-gtk-module

安装 wine（推荐：挂代理安装）

根据 winehq官方文档 进行操作。

  1. 如果系统时 64 位，需要执行下面的命令
sudo dpkg --add-architecture i386
  1. 下载并添加存储库密钥：
wget -nc https://dl.winehq.org/wine-builds/winehq.key
sudo apt-key add winehq.key
  1. 添加存储库：
sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ eoan main'
  1. 更新包：
sudo apt update
  1. 安装 wine
sudo apt install --install-recommends winehq-stable
  1. 安装 wine-gecko、wine-mono
# 下载软件包
http://dl.winehq.org/wine/

# 复制文件到  ~/.wine/drive_c


# 手动安装（推荐自动下载安装）
wine msiexec /i wine-gecko.msi
wine msiexec /i wine-mono.msi
