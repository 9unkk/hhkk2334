SQL注入简介

SQL注入英文名叫 SQL Injection，是发生于应用程序与数据库层的安全漏洞。简单来说，就是在输入的字符串中注入 SQL 命令，如果程序忽略了字符检查，那么这些注入进去的恶意软件就会被数据库误认为事正常的 SQL 命令来运行。

http://192.168.150.128/sqli-labs-master/Less-1/?id=1' order by 4%23

这里的 %23 表示的是 “#” 符号，用来注释后面的语句。字符 “#” 浏览器不会编码，需要手动编码 %23
